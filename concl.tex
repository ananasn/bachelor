\conclusion
В выпускной квалификационной работе был реализован программный комплекс автоматизированной подготовки управляющих программ для гравировальных станков с ЧПУ. При разработке программного комплекса был применен новый подход к генерации управляющих программ. Предлагаемый подход обеспечил возможность генерировать управляющие программы на основе растровых изображений, имея только выход в~\mbox{интернет}. А программный комплекс, разработанный для поставленной цели, оказался лучше аналогичных программных продуктов по большинству показателей качества.

Для достижения поставленной цели были решены следующие задачи:
\begin{itemize} 
\item Проведен сравнительных анализ современных систем автоматизированной поготовки управляющих программ для гравировальных станков и \mbox{предложенного подхода}.
\item Разработан алгоритм обработки растрового изображения, с целью получения 3D-модели поверхности.
\item Найден \mbox{наилучший} формат 3D-моделей для применении в~\mbox{веб-сфере}.
\item Реализован модуль редактирования изображений.
\item Реализован модуль генерации УП.
\item Разработка опробована на конкретных примерах.
\end{itemize}
