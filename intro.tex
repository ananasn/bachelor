Технологическая подготовка производства\,(ТПП) является одним из важнейших этапов жизненного цикла изделия.
Она обеспечивает технологическую готовность производства и оказывает значительное влияние на себестоимость конечной продукции. ТПП включает в \mbox{себя} \mbox{довольно} разнообразный спектр задач, решаемых технологом, одной из ниx    \mbox{является} подготовка управляющих программ\,(УП) для станков с~ЧПУ. Стоит \mbox{отметить}, что на реальном производстве практически не бывает случаев, когда УП возможно написать вручную в \mbox{силу} их высокой сложности и~\mbox{объема}. Процесс гравировки не \mbox{является} \mbox{исключением}~--~УП, \mbox{необходимая} для \mbox{нанесения} растрового изображения, состоит из множества \mbox{однотипных} G-кодов, чье количество может исчисляться \mbox{тысячами}.

На данный момент не так много систем автоматизации, специализирующихся на гравировке. Большинство таких программных \mbox{пакетов} поставляются к конкретным станкам, из-за чего эти системы \mbox{являются} неуниверсальными. Стоит заметить, что гравировка может \mbox{выполняться} на универальном обородувании, а к нему программное обеспечение, необходимое для нанесения изображения, может не прилагаться вовсе. Программные системы, которые не  привязаны к~конкретному оборудованию представляют собой  массивные десктопные приложения, требующие установки и занимающие большое количество вычислительных \mbox{ресурсов}. Таким образом, очевидно, что вопросы реализации и построения \mbox{систем} \mbox{автоматизированной} подготовки управляющих программ для гравировки {\bfseries \mbox{исследованы} в~недостаточной степени}.

Все вышеперечисленное указывает на {\bfseries актуальность} рассматриваемой темы и~ доказывает необходимость применения иного подхода, а~\mbox{именно}~--~применение веб-технологий.
~\\

\begin{samepage}{\bfseries Объектом исследования} \mbox{является} система автоматизированной подготовки управляющих программ для гравировальных станков с ЧПУ.\end{samepage}

В~\mbox{качестве} {\bfseries предмета исследования} рассматриваются методики, \mbox{алгоритмы} и {инструменты}, применяемые для построения таких систем. 

{\bfseries Целью} данной работы является реализация программного комп\-лекса автоматизированной подготовки управляющих программ для  гравировальных станков с ЧПУ. Для достижения поставленной цели были выделены следующие {\bfseries задачи}:
\begin{itemize} 
\item Провести сравнительных анализ современных систем автоматизированной поготовки управляющих программ для гравировальных станков и \mbox{предложенного подхода}.
\item Разработать алгоритм обработки растрового изображения, с целью получения 3D-модели поверхности.
\item Сравнить характеристи форматов 3D-моделей, для поиска наилуч\-шего при применении в веб-сфере.
\item Реализовать модуль редактирования изображений.
\item Реализовать модуль генерации УП.
\item Опробовать разработку на конкретных примерах.
\end{itemize}

Для решения данных задач было решено применить следующие {\bfseries инструменты и технологии}: веб-технологи \cite{web_book}, технология разработки веб-приложений с~помощью микрофреймворка Flask \cite{Flask_link, Flask_book}, основые положения функционирования клиент-серверной архитектуры \cite{Pop20141172}, технология отображения 3D-графики в браузере с помощью библиотеки Three.js \cite{Evans20144, Threejs_link}, инструменты обработки и редактирования изображений \cite{Canvas_book, JS_book}, правила построения управляющих программ для станков с ЧПУ \citep{G_book}.

{\bfseries Научная новизна} работы состоит в предложенном подходе генера\-ции управляющих программ для гравировальных станков с ЧПУ с~исполь\-зованием веб-технологий.

Также стоит отметить, что автоматизация получения УП имеет значительную {\bfseries практическую ценность} не только для работ, связанных  с~декорированием, часто выполняющихся мелкими частными фирмами, но и для крупных предприятий. Примером могут служить гравировка \mbox{дорожек} на печатных платах, нанесение символов на клавиши управляющих стендов и лицевые поверхности приборов.

{\bfseries Структура} выпускной квалификационной работы. Выпускная квалификационная работа состоит из введения, 3 глав, заключения, библиографии и 6 приложений.

{\bfseries Во введении} рассматриваются актуальность темы работы, степень ее исследования, а также объект и предмет исследования. Далее сформулированы цель и задачи работы. Указаны инструменты и технологии применяемые для реализации.

{\bfseries В первой главе} рассматриваются существующие программные продукты, проводится их сравнительный анализ.

{\bfseries Во второй главе} рассмотрены модули реализуемого программного комплекса. Перечень модулей включает в себя: модуль построения 3D-модели \mbox{рельефа}, модуль редактирования изображений, модуль генерации G-кодов.

{\bfseries В третьей главе} демонстрируется работа разработанного программного комплекса автоматизированной подготовки УП для гравировальных станков с ЧПУ.

{\bfseries В заключении} приводятся результаты и выводы.
