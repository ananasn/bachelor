\chapter{Руководство оператора}
\section{Назначение}
Программный комплекс автоматизированной подготовки управляющих программ для гравировальных станков с ЧПУ \mbox{предназначен}~для:
\begin{itemize}
 \item генерации управляющих программ на основе растровых \mbox{изображений};
 \item генерации 3D-моделей рельефов на основе растровых изображений;
 \item просмотра генерируемых 3D-моделей рельефов;
 \item редактирования изображений.
 \end{itemize}
 \section{Условия выполнения}
 Для работы программного комплекса требуется выполнение следующих условий:
 \begin{itemize}
 \item наличие оборудования с подключением к сети;
 \item наличие веб-браузера с поддержкой технологий WebGL и Canvas.
 \end{itemize}

 \section{Выполнение}
Работа с программным комплексом начинается с открытия главной страницы\,(\autoref{fig:index}). Для выбора и загрузки изображения требуется \mbox{нажать} на кнопки $1$ и $2$ соответственно, при этом надпись <<файл не \mbox{выбран}>> поменяется на название загруженного файла. Далее изображение можно отредактировать, нажатие на кнопку $3$ перенаправляет пользователя на страницу редактирования (\autoref{fig:ed}). Для того, чтобы скачать 3D-модель рельефа и УП, нужно нажать на кнопки $5$ и $6$ соответственно. Кнопка $4$~перенаправляет нас на просмотрщик 3D-моделей (\autoref{fig:v}).
